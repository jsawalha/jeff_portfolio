Project # 1: Individualized identification of first-episode bipolar disorder using machine learning and cognitive tests

December 18, 2020

This was a publication examining neurocognitive data in patients with bipolar I disorder (BD). We wanted to examine the correlation between certain neurocognitive deficits and the number of manic / psychotic episodes in first-episode and chronic-BD participants (FE-BD & CHR-BD). The overall objective of this study was to identify individuals who are in the beginning stages of bipolar disorder, by learning the more nuanced cognitive deficits that CHR-BD participant’s display in their neurocognitive tests. With the help of ML models, we were successfully able to determine which individuals resembled a more progressive stage of BD versus a control group. We were also able to determine exactly which cognitive domains were most affected by manic/psychotic episodes.

Here is a list of skills I deployed for this publication:

  • Used python to build a machine learning pipeline for 2 separate datasets using pandas, numpy, and sklearn
  • Used imputation methods (sklearn) to fill missing data for critical patients and control participants
  • Tried a series of normalization functions to reduce variance in model evaluation
  • Used my own nested cross validation function to hyper-tune parameters of a support vector machine model
  • Used seaborn and matplotlib to create visualizations for final publication
  • Examined each individual prediction to examine if mood-state was a latent factor in predictions
