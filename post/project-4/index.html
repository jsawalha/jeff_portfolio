Project # 4: Detecting presence of PTSD using sentiment analysis from text data

April 12, 2017

Sentiment analysis refers to the use of natural language processing to extract emotional content from text information. In our paper, we train a machine learning model on textual data, to identify individuals with PTSD using sentiment analysis from semi-structured interviews. Our sample size included 188 individuals without PTSD, and 87 with PTSD. The interview was conducted by an artificial character (Ellie) over a video-conference call. Our model was able to achieve a balanced accuracy of 80.4% on a held out dataset used from the AVEC-19 challenge. Additionally, we implemented various partitioning techniques to determine if our model was generalizable enough. This shows that learned models can use sentiment analysis of speech to identify the presence of PTSD, even through a virtual medium. This can serve as an important, accessible and inexpensive tool to detect mental health abnormalities during the COVID-19 pandemic.

Here is a list of skills I deployed for this publication:

  • An entire data pipeline dedicated to feature engineering sentiment scores for each individuals utterance in the interviews. This included libraries such as NLTK, pandas, numpy, sklearn, and imblearn
  • Several differing partitioning methods to test the generalizability of our models
  • Tried 3 different sentiment analyzers, all of which needed different pre-processing steps
  • Attempted using Bidirectional Encoder Representations from Transformers (BERT) to classify PTSD from Non-PTSD individuals
  • Using parametric tests to compare the distribution of negative and positive sentiments between both groups
